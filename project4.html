📱 Placement Training – Mobile Application

📖 Introduction

The Placement Training Mobile Application is an Android-based platform built to help students prepare for job placements and interviews. It offers interactive learning in areas like general aptitude, verbal reasoning, technical MCQs, and programming concepts. The app also features timed online tests for practice and self-evaluation.

🛠️ Technologies Used

  • Java – Android Application Logic
  • XML – User Interface Design
  • SQLite – Local Database for Questions and Scores
  • Android Studio – IDE

🔍 App Features

  • User Registration and Login
  • General Aptitude and Verbal Reasoning Modules
  • Concept-based MCQs and Quizzes
  • Technical and Programming Practice
  • Online Tests with Timer
  • Score Tracking and Feedback

🚀 How to Run

- Open the project in Android Studio
- Connect a real device or emulator
- Build and run the application
- Use the sample login or register as a new user to explore the modules
