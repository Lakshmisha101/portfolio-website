🔧 Automated Fault Detection in Turbocharger Using Machine Vision

📖 Project Overview

As part of my internship, I developed an automated real-time fault detection system using machine vision techniques. The system inspects turbocharger components using a camera and machine learning model to identify missing or defective parts during production, enhancing the accuracy and efficiency of quality control in manufacturing.

🛠️ Technologies Used

  • Python – Core development and model integration
  • OpenCV – Real-time image processing
  • Machine Learning – Trained model for component detection
  • Camera Integration – Live image capture for inspection
  • NumPy, Pandas – Data analysis and model preprocessing

🔍 Key Features

  • Real-time visual inspection of turbocharger units
  • Defect classification (e.g., missing bolts, cracks)
  • Alerts and logs generated on detection of faults
  • Minimizes human error in manual inspection
  • Adaptable to other mechanical parts inspection

📸 Project Screens / Photos

🚀 How It Works

- The camera captures frames of turbochargers on the assembly line
- Each frame is analyzed in real time by a trained ML model
- If a missing component or defect is detected, the system highlights the issue
- A report is logged, and optional alerts are raised to quality control
- Data is stored for further analysis and system improvement

🏭 Outcome & Impact

  • Reduced fault detection time
  • Improved production line quality assurance
  • Lower defect leakage to final assembly
  • Real-time feedback for industrial process control
