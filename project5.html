🌾 Cotton Pest Trap Using Machine Learning Models

📖 Project Overview

This project implements an advanced pest detection system for cotton fields using YOLOv8, machine learning, and IoT technology. The system enables real-time identification and tracking of pest-infested areas, contributing to sustainable pest control and improved crop protection.

🛠️ Technologies Used

  • Python – For model training and data processing
  • YOLOv8 – Object detection for identifying pests
  • ESP32-CAM – IoT camera module for live field surveillance
  • Internet of Things – Real-time communication and deployment
  • OpenCV – Image annotation and preprocessing

🔍 Key Features

  • Real-time pest detection and classification
  • Automated image annotation using custom datasets
  • IoT-based deployment with ESP32-CAM
  • Alert system for farmers to respond to infestations
  • Data logging and image storage for analysis

📸 Project Screens / Photos

🚀 How It Works

- The ESP32-CAM captures live images from cotton fields
- YOLOv8 processes these images to detect pests
- Alerts are generated if pest presence crosses a threshold
- Images and data are stored for analysis and future training
- Farmers can be notified via a mobile app or SMS gateway

🌱 Outcome & Benefits

  • Improved early pest detection
  • Reduced pesticide overuse
  • Higher cotton yield and productivity
  • Low-cost, scalable solution for rural farms
