💱 Currency Converter Web App

🌍 About the Project

This Currency Converter Web App allows users to convert currencies based on real-time exchange rates. It fetches live data from public APIs and displays results along with country flags.

🛠️ Features

  • Real-time currency conversion using APIs
  • Convert any amount between two currencies
  • Displays country flags for selected currencies
  • Built using JavaScript Promises and API calls
  • Responsive and user-friendly UI

🔗 APIs Used

  • Flags API: https://flagsapi.com/${countryCode}/flat/64.png
  • Currency Exchange API: https://latest.currency-api.pages.dev/v1/currencies

🚀 How It Works

  1. User selects input and output currencies.
  2. User enters amount to convert.
  3. App fetches exchange rates from API.
  4. Displays converted amount and country flags.

🖥️ Technologies Used

  • HTML – For structure
  • CSS – For styling and layout
  • JavaScript – For API logic and functionality
