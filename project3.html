🌐 Travel Website – DBMS Project

📖 About the Project

This Travel Website DBMS Project is designed to handle various travel-related data and functionalities like user management, destination listings, bookings, and admin controls. It integrates a front-end interface with a MySQL back-end to demonstrate real-world database-driven web application development.

🛠️ Features

  • User registration and authentication
  • Destination search and filtering
  • Hotel and package booking system
  • User reviews and rating support
  • Admin portal for managing packages and users
  • Responsive and dynamic UI

🧩 Modules & Pages

  • Login Page
  • Home Page
  • Destination Package Display
  • Hotel Booking Page
  • Admin Login and Admin Dashboard
  • Customer Package Details View
  • About Us Page

🗂️ Database Design

Entities used in the database:

  • Users
  • Destinations
  • Hotels
  • Activities
  • Reviews
  • Bookings

Entity-Relationship diagrams (ERDs) are used to design the relationships between these entities.

💻 Technologies Used

  • HTML, CSS, JavaScript – Front-End
  • PHP – Back-End
  • MySQL – Database
